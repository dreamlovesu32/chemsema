# Reconstruction Principles

ChemCore OCR is inverse document rendering. The output should be a ChemCore
command stream and internal JSON, not an image trace.

## Primary Invariants

- Structure first, pixels second.
- Ground truth is ChemCore internal molecule JSON or selected fragment JSON.
- Compare topology, node labels/source text, bond order, variant, stereochemical
  direction, double placement, and label attachment.
- Ignore id ordering, command ordering, atom ordering, and small layout noise.
- Do not ignore large coordinate drift, wrong label style, wrong attachment, or
  wrong bond geometry.

## Geometry Recovery

Recover atom sites before labels:

- bond endpoints from stroke centerlines and endpoint support
- junctions from line extension and intersection
- ring vertices from consistent polygon edge geometry
- template centers from stable ring centerlines when shape evidence is strong
- label attachment from recovered atom sites and visible text relation

Hydrogen glyphs generated by labels are not anchors except standalone `H`.

## Style Inference

Infer global style from measured image evidence:

- body text height, especially molecule labels, commonly anchors 10 pt
- bond length from the majority of clean single bonds
- bond width from local stroke width
- double-bond spacing from observed parallel stroke separation
- wedge/hash/wavy from contour and stroke family evidence

Use ACS template defaults only for unmeasured rendering parameters such as label
retreat when the source image does not provide direct evidence.

## Rule Hygiene

A valid rule should explain a structural phenomenon:

- endpoint-pair support creates one bond
- parallel compatible evidence upgrades bond order
- generated hydrogens cannot anchor bonds
- visible label reversal maps to source text through ChemCore
- aromatic crossing bonds may imply hidden ring completion only when occlusion
  makes direct recovery impossible

Avoid rules that say "because label X appears, move bond Y". That is a corpus
patch, not an OCR rule.

## Development Loop

Always inspect JSON before rendered comparison. Visual output can reveal layout
defects, but JSON reveals whether the reconstructed molecule is the same
editable object.
