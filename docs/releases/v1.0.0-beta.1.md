# ChemSema v1.0.0-beta.1

This is the first public beta release of ChemSema, an open-source chemistry
structure editor focused on CDXML compatibility, daily research drawing, and
Office copy/paste workflows.

## Highlights

- Shared Rust engine for document state, editing commands, hit testing,
  chemistry label rules, implicit hydrogens, render primitives, and CDXML/CDX
  import/export.
- Browser viewer backed by the same WASM engine used by the desktop surface.
- Windows desktop and Office/OLE foundations for native file handling,
  clipboard payloads, SVG/EMF export, and Word-oriented editable paste.
- Published-figure comparison assets using maintainer-authored CDXML files.
- Synthetic CDXML fixtures and SVG golden snapshots for CI-friendly regression
  testing.
- GitHub Actions CI and GitHub Pages demo deployment workflows.

## Current Limitations

- Windows installer packaging is still under testing.
- ChemDraw oracle comparison scripts require a local Windows ChemDraw
  installation and run as optional local Windows validation.
- Office/OLE integration is Windows-specific.

## Validation

The release is validated with:

```bash
npm ci
cargo test
npm run verify
```
