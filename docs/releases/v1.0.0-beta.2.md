# ChemCore v1.0.0-beta.2

This beta focuses on label semantics, chemistry-summary correctness, and a
small context-menu rendering fix discovered during real desktop use.

## Highlights

- Generic labels now keep chemistry summaries honest. If a selected molecule
  contains `R`, `R'`, `R''`, or connected `Ar`, ChemCore does not show formula
  or molecular-weight values because the composition is intentionally
  indeterminate.
- Connected `Ar` is recognized as a generic aryl label during structure-label
  editing instead of being treated as argon. Explicit element replacement
  remains available through the element tool workflow.
- Context-menu check and submenu indicators no longer use corrupted text
  glyphs, avoiding mojibake in the right-click menu.
- The browser WASM engine and Windows desktop executable were rebuilt from the
  same corrected Rust engine.
- Regression tests now cover indeterminate generic labels while preserving
  complete abbreviation expansion summaries.

## Validation

The release is validated with:

```bash
npm ci
cargo test
npm run verify
```

The Windows installer also includes the rebuilt desktop executable and the
Office/OLE integration service used by the desktop workflow.
