# ChemSema v1.0.0-beta.3

这个 beta 是 Windows 安装包的热修复版本，重点修复 Office/OLE 注册步骤。
它延续 beta.2 的编辑器和内核范围，但修掉了一个打包路径回归：桌面文件已经解压完成后，安装器可能在 Office/OLE 注册阶段中断。

## 重点

- NSIS 安装器现在会先在实际安装目录中查找 `chemsema-office.exe`，再 fallback 到旧的 `resources` 子目录布局。
- Office/OLE 注册会先尝试 machine-wide COM 注册；如果这一步无法启动或返回失败码，安装器会自动降级为当前用户注册，而不是直接中止整个安装。
- 卸载 hook 现在会同时尝试 machine-wide 和 current-user 反注册，让失败安装或混合注册范围的安装更容易清理干净。
- Windows x64 安装包已经重新构建，并在清理旧安装目录、注册表、AppData、快捷方式和临时文件后完成手动干净安装验证。

## 验证

本 release 使用以下命令验证：

```bash
npm ci
cargo build -p chemsema-office --release
npm run desktop:build
```

Windows 手动验证覆盖了从生成的 NSIS 安装包干净安装、Office/OLE 注册、安装痕迹清理，以及从下载目录安装包再次安装。

## 发布状态

这个版本仍然是 beta。安装包目前还没有代码签名，因此 Windows 可能会显示 SmartScreen 提醒。正式 `v1.0.0` 建议等路线图里的打包、签名、升级/卸载和 Office 工作流验证条件继续收敛后再发布。
