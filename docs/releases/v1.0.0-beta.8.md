# ChemCore v1.0.0-beta.8

This release aligns ChemCore's chemical-label interpretation, CDX/CDXML
round trips, attachment geometry, and short-bond rendering more closely with
ChemDraw while keeping native CCJS/JSON semantic and readable.

## Highlights

- Treats formula groups such as `C10H21` as structured chemical tokens and
  reverses multi-group formulas by group without losing source text or numeric
  subscripts.
- Preserves CDX/CDXML rich-text runs, standard colors, explicit hydrogen
  counts, label display direction, and `BeginAttach`/`EndAttach` semantics
  across stable open-save-open cycles.
- Uses explicit imported label attachments when available and deterministic
  structural anchors otherwise; per-glyph clipping prevents parentheses,
  descenders, and script text from shifting unrelated bond anchors.
- Corrects centered and side double-bond retreat, including equal visible
  centered lines and exact-tie side placement after adding a substituent.
- Matches ChemDraw's adaptive three-, two-, and one-segment distribution for
  very short dashed bonds while preserving the underlying bond semantics.
- Rebuilds the shared engine for Web, Windows desktop, HarmonyOS, native CLI,
  and Codex skills. The CLI skill now includes verified Windows x64 and Linux
  x64 runtimes, with Ubuntu/WSL and Ubuntu CI coverage.
- Refreshes the public editor screenshot and published CDXML comparison SVG
  and Office EMF assets with the beta.8 engine.

## Validation

The release was validated locally with the locked Rust workspace tests, full
GUI regression, viewer and label-anchor regressions, Windows desktop/Office/CLI
release builds, Web/Harmony WASM identity checks, an unsigned Harmony HAP build,
and native Ubuntu/WSL engine and CLI tests.
