# ChemCore v1.0.0-beta.2

这个 beta 版本主要修复真实桌面使用中发现的标签语义、化学摘要正确性，以及右键菜单显示问题。

## 重点

- 泛基团标签现在不会给出误导性的化学摘要。选中分子中含有 `R`、`R'`、`R''` 或已连接 `Ar` 时，ChemCore 不显示分子式或分子量，因为这类组成在语义上是不确定的。
- 已连接的 `Ar` 在结构标签编辑中识别为芳基泛基团，不再当作氩元素处理。显式元素替换仍通过元素工具链路完成。
- 右键菜单的勾选与子菜单指示符不再使用损坏字符，避免出现乱码。
- 浏览器 WASM engine 和 Windows 桌面端可执行文件已基于同一套修正后的 Rust engine 重建。
- 增加泛基团标签的化学摘要回归测试，同时保持完整缩写展开时的分子式/分子量摘要能力。

## 验证

本 release 使用以下命令验证：

```bash
npm ci
cargo test
npm run verify
```

Windows 安装包包含重建后的桌面端可执行文件，以及桌面工作流使用的 Office/OLE 集成服务。
