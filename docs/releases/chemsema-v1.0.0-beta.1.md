# ChemSema 1.0.0-beta.1

This is the first public beta under the ChemSema name. It keeps the existing
Git history while completing the rename across the repository, Rust crates,
desktop and Office applications, CLI, agent skills, documentation, generated
WASM bindings, and public URLs.

Highlights:

- One consistent `ChemSema` / `chemsema` identity across product and code.
- Current editor, desktop, Office/OLE, CLI, and agent workflows from the prior
  beta line, including the latest chemical-label display refinements.
- Permanent compatibility handling for the prior GitHub repository and Pages
  links, with local and scheduled monitoring.
- Fresh Windows x64 installer and CLI artifacts carrying the ChemSema identity.

The Windows installer is not code-signed. Windows may show a SmartScreen
warning during installation.
