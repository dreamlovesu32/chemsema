# ChemSema v1.0.0-beta.1

这是 ChemSema 的第一个公开 beta release。ChemSema 是一个开源化学结构编辑器，重点面向 CDXML 兼容、日常科研绘图和 Office 复制粘贴工作流。

## 重点

- 共享 Rust engine 负责文档状态、编辑命令、命中测试、化学标签规则、隐式氢、render primitives 和 CDXML/CDX 导入导出。
- 浏览器端使用同一套 WASM engine，和桌面端共享核心行为。
- Windows 桌面端与 Office/OLE 基础链路已经建立，包括原生文件处理、剪贴板 payload、SVG/EMF 导出，以及面向 Word 的可编辑粘贴。
- 使用维护者本人绘制的 CDXML 文件生成真实论文图对比资产。
- 添加 synthetic CDXML fixtures 和 SVG golden snapshots，作为适合 CI 的回归测试。
- 添加 GitHub Actions CI 和 GitHub Pages demo 部署 workflow。

## 当前限制

- Windows 安装包仍在测试中。
- ChemDraw oracle 对比脚本需要本机 Windows ChemDraw 安装，作为可选本地 Windows 验证流程运行。
- Office/OLE 集成是 Windows 专属能力。

## 验证

本 release 使用以下命令验证：

```bash
npm ci
cargo test
npm run verify
```
