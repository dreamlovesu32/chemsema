# ChemCore v1.0.0-beta.8

这个版本进一步对齐 ChemDraw 的化学标签解释、CDX/CDXML 往返、连接锚点几何和
短键渲染，同时保持原生 CCJS/JSON 的语义完整与可读性。

## 重点

- 将 `C10H21` 等分子式组合解释为结构化化学 token；多组合分子式按组翻转，
  同时保留 source text 和数字下标。
- 稳定保留 CDX/CDXML 富文本 run、标准颜色、显式氢数量、标签显示方向以及
  `BeginAttach`/`EndAttach` 语义，确保 open-save-open 往返一致。
- 有显式标签锚点时读取并使用；缺失时回退到确定性的结构锚点。逐 glyph 裁切避免
  括号、下伸部和上下标错误拉低无关字符的键锚点。
- 修复中心双键和侧双键退让，包括中心双键最终等长，以及新增取代键产生精确侧别
  平局时的自动换边。
- 对齐 ChemDraw 对极短虚键从三段、两段到一段的自适应分布，同时保持底层键语义。
- 重新构建 Web、Windows 桌面端、HarmonyOS、原生 CLI 和 Codex skills 共用的内核。
  CLI skill 现在同时携带已校验的 Windows x64 与 Linux x64 运行时，并加入
  Ubuntu/WSL 和 Ubuntu CI 验证。
- 使用 beta.8 内核刷新公开编辑器截图、论文 CDXML 对比 SVG 和 Office EMF 资产。

## 验证

本版本在本地完成 locked Rust workspace 测试、完整 GUI 回归、viewer 与标签锚点
回归、Windows 桌面/Office/CLI release 构建、Web/Harmony WASM 一致性检查、
Harmony unsigned HAP 构建，以及 Ubuntu/WSL 原生 engine 与 CLI 测试。
