# ChemCore v1.0.0-beta.4

这个 beta 是一次覆盖面较大的交互与自动化版本。它改进了大文件编辑、CDXML 渲染/导入保真、hover 和拖拽反馈、浏览器文件处理、回归覆盖，并新增 headless CLI，让 agent 和脚本可以在不驱动 GUI 的情况下调用 ChemCore。

## 主要变化

- 新增 `chemcore-cli`，支持 headless 文档检查、格式转换、导出、空白文档生成和 JSON 命令执行。
- 新增适合 agent 调用的直接 engine 命令，覆盖文本、标签、目标变换、删除、箭头几何、图形几何、文档 load/export/convert/inspect 和文档样式应用。
- 新增结构化 CLI 报告，包含每条命令的 `ok`、`executed`、`changed`、新建/更新/删除 id、错误信息、after 快照和最终文档状态。
- 新增 `--document-json` 和 `--inspect-after`，自动化流程可以在命令后或部分失败后读取 ChemCore 内部 JSON、分子状态和对象状态。
- 改进 CDXML 导入导出保真度，覆盖导入标签、箭头、符号、粗线宽度、自由基价态、grouped arrows、堆叠标签、attached-label group layout、数字 glyph 锚点、标签内部缓存 fragment，以及带括号的 sulfonyl 标签。
- 重做标签裁剪和 glyph 几何，更新 glyph clip polygon 覆盖，并刷新公开 synthetic SVG 快照。
- 改进大文件交互性能：局部化渲染更新、缓存拖拽预览输入、减少整页刷新路径、优化对象创建延迟，并在 patch 前同步 deferred object creation。
- 重建大结构拖拽预览和选择管线，包括本地绘制预览、frame-local structure drag preview、后端 target primitives、前端 partial-bond preview，以及更稳定的 live selection preview。
- 修复绘制、pointer commit、对象创建、选择拖拽、已选对象拖拽、括号 handle 编辑、箭头 handle 编辑和多分子拖拽中的 hover、focus 与 overlay 残留。
- 改进 grouped-object 编辑语义、grouped molecule hit testing、框选行为，以及多分子拖拽的增量渲染 target。
- 优化箭头、括号、图形和对象控制点，包括弯箭头预览、括号 resize refresh、统一控制点样式，以及拖拽选择时隐藏诊断 marker。
- 新增浏览器文件拖拽打开、当前 viewer 打开文件、shared display scale 处理，以及更快的桌面端/viewer 开发脚本。
- 增加 viewer interaction、大文件拖拽预览、大对象操作、glyph clip manifest、SVG 像素对比和 PowerPoint/CDX 渲染对比相关回归与诊断流程。
- 新增中英文 CLI 命令指南、公开交互反馈规则、早期项目历史说明，并更新 README 架构说明，明确共享 Rust engine、对人友好的 UI 和对 agent 友好的 CLI。

## 说明

桌面 UI 和 CLI 共享同一套 engine 和文档模型。UI 继续面向科研人员直接绘图和编辑；CLI 面向确定性的文件和命令工作流，调用方直接提供坐标、id 和 JSON 命令，不需要模拟 GUI 交互。
