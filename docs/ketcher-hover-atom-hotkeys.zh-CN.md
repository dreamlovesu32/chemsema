# Ketcher 悬浮端点快捷键规则

鼠标悬浮到端点后，不点击，直接按键：

| 按键 | 结果 |
| --- | --- |
| `h` | `H` |
| `c` | 清除掉，变成默认碳 `C` |
| `n` | `N` |
| `o` | `O` |
| `s` | `S` |
| `P` | `P` |
| `p` | `Ph` |
| `f` | `F` |
| `l` | `Cl` |
| `b` | `Br` |
| `i` | `I` |
| `m` | `Me` |
| `S` | `Si` |
| `N` | `Na` |
| `B` | `B` |
| `d` | `D` |
